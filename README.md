# Función double:
Recorre el arreglo de uno en uno usando un ciclo y va multiplicando cada elemento por 2, modificando su valor dentro del mismo arreglo.

# Función maximum:
Se declara una variable maximum igual al valor del primer elemento del arreglo, luego recorre el arreglo con un ciclo y si el valor que se encuentra en el indice es mayor al valor de la variable maximum, ahora ese valor se almacena en maximum. 

# Función average:
Retorna el cociente entre suma de todos los elementos del arreglo y el total de elementos del mismo. Se usa la función sum propia de python para sumar los elementos del arreglo.
